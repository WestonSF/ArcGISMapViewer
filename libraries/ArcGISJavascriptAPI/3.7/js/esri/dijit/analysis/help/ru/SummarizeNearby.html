Суммировать близлежащие

Этот инструмент находит объекты, которые расположены в пределах указанного расстояния от объектов на слое анализа. Расстояние может быть измерено по прямой, по времени в пути (например, в пределах 10 минут) или по расстоянию пути (в пределах 5 км). Затем вычисляется статистика для близлежащих объектов. Например:

  • Вычисляет численность населения в пределах пяти минут езды от предполагаемого местоположения нового магазина.
  • Вычисляет число съездов с автотрассы в пределах одной мили вдоль дороги от предполагаемого местоположения нового магазина для использования в качестве меры доступности.

Если опция Использовать текущий экстент карты (Use current map extent) включена, будут анализироваться только те объекты слоя анализа и слоя объединения, которые видны в момент анализа в текущем экстенте. Если опция отключена, будут анализироваться все объекты слоя анализа и слоя объединения, даже если они находятся вне текущего экстента карты.

Выберите слой для суммирования

Объекты этого слоя, находящиеся в пределах указанного расстояния от объектов в слое анализа, будут суммированы.

Найти ближайшие объекты, используя

Вы можете определить, какой тип измерения расстояния вы хотите использовать: расстояние по прямой, расстояние по пути или время в пути. Расстояние пути и время пути используют дорожную сеть и учитывают различные ограничения, такие как улицы с односторонним движением. Время пути также учитывает известные ограничения скорости. Вы можете задать несколько расстояний. Объекты, находящиеся в пределах указанных расстояний (или на равным им) будут суммироваться.

Добавить статистику из

Дополнительно вы можете вычислить статистику любых числовых атрибутов ближайших точек.

При включении опции Число точек (Count of points) будет вычислено общее число ближайших точек. Кроме того, вы можете вычислить статистику числовых атрибутов ближайших точек.

  • Сумма (Sum) – суммарное значение указанного поля
  • Среднее (Average) – вычисляет среднее арифметическое указанного поля
  • Минимум (Minimum) – определяет минимальное значение указанного поля
  • Максимум (Maximum) – определяет максимальное значение указанного поля
  • Стандартное отклонение (Std. Deviation) – стандартное отклонение для указанного поля

Выбрать поле для группировки

Это атрибут ближайших объектов, который может использоваться для вычисления статистики отдельно для каждого уникального значения атрибута. Например, предположим, что ваш слой анализа содержит свободные земельные участки, предназначенные для продажи, а ближайшими объектами являются контуры зданий. Одним из атрибутов контуров зданий является UseType (тип использования), имеющий значения RETAIL (магазин), RESIDENTIAL (жилое), MANUFACTURING (производство), PUBLIC (общественное) и т.д. Чтобы охарактеризовать типы зданий вблизи каждого свободного участка, вы можете вычислить общую площадь для каждого уникального типа зданий в пределах двух километров от свободных участков, используя UseType как группировку по атрибуту.

Имя слоя результата

Это имя слоя, который будет добавлен в Мои ресурсы (My Contents) и на карту. Имя слоя по умолчанию базируется на имени слоя анализа. Если такой слой уже существует, появится запрос на его перезапись.

С помощью раскрывающегося поля Сохранить результат в (Save result in) вы можете задать имя папки в разделе Мои ресурсы (My Contents), в которую будет записан результат.
