데이터 추출

이 도구를 사용하면 지정된 관심 영역에 대한 데이터를 선택하여 다운로드할 수 있습니다.선택한 레이어는 zip 파일 또는 레이어 패키지에 추가됩니다.

추출할 레이어

피처를 추출할 수 있는 컨텐츠 테이블에 있는 레이어입니다.선택한 레이어가 추출되어 결과 zip 파일 또는 레이어 패키지에 포함됩니다.

추출할 레이어 목록에 레이어를 표시하려면 데이터를 소유하거나 데이터 소유주가 데이터를 내보내도록 허용해야 합니다.소유한 데이터를 내보내는 방법:

  • 데이터의 항목 설명을 여세요.
  • 항목 설명을 편집하려면 편집을 클릭하세요.
  • 등록정보 섹션이 나올 때까지 화면을 아래로 스크롤하세요.
  • 데이터 내보내기에 체크하세요.

처리 범위

출력 zip 파일 또는 레이어 패키지에 포함할 피처를 정의하는 영역입니다.다음 중에서 선택할 수 있습니다.

  • 맵의 현재 가시 범위( 표시 범위와 동일함)
  • 컨텐츠 테이블에 있는 레이어 중 하나
  • 처리 범위를 정의할 영역 정의
추출할 맵 범위 내에 완전히 포함되어 있는 모든 피처.완전히 범위 내에 있지 않고 맵 범위 경계를 가로지르거나 교차하는 피처의 경우 다음 두 가지 옵션 중에서 선택할 수 있습니다.
  • 피처 선택 - 처리 범위의 경계를 가로지르는 전체 피처가 추출됩니다.
  • 클립 피처 - 처리 범위 내에 있는 피처 부분만 추출됩니다.

출력 데이터 형식

추출 및 다운로드할 데이터의 형식입니다.레이어 패키지는 항상 파일 지오데이터베이스를 포함합니다.

출력 파일 이름

내 컨텐츠에 추가되는 항목의 이름입니다.항목 이름은 선택한 출력 데이터 형식을 기반으로 합니다.항목이 이미 있는 경우 덮어쓸지 확인하는 메시지가 나타납니다.

결과 저장: 드롭다운 상자를 사용하여 내 컨텐츠에서 결과를 저장할 폴더의 이름을 지정할 수 있습니다.
