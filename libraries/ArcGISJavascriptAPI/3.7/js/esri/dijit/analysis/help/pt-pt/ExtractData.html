Extrair dados

Com esta ferramenta, é possivel selecionar e transferir dados de uma área específica de interesse. As camadas serão adicionadas a um ficheiro zip ou a um pacote de camadas.

Camadas a extrair

As camadas na tabela de conteúdos a partir das quais pode extrair elementos. Camadas seleccionadas e incluídas num ficheiro zip ou resultante de um pacote de camadas.

Para que uma camada apareça na lista de camadas a extrair, deve possuir os dados ou o proprietário dos dados deve permitir exportação dos mesmos. Para permitir a exportação dos seus dados:

  • Abra a descrição do item dos dados.
  • Clique em EDITAR para alterar a descrição do item.
  • Navegue até Propriedades.
  • Ative Exportar Dados.

Área de estudo

Área que define quais os elementos que serão incluídos no ficheiro zip ou no pacote de camadas. Pode escolher entre os seguintes:

  • Extensão atual do mapa ( Igual à exibida).
  • Uma das camadas da tabela de conteúdos.
  • Desenhar uma área de forma a definir a área de estudo.
Todos os elementos que se encontram inseridos na extensão do mapa serão extraídos. Para os casos em que os elementos se cruzam ou intersetam com os limites da extensão do mapa mas que não estão completamente inseridos na extensão, tem duas opções:
  • Selecionar os elementos-Os elementos que cruzam com os limites da área de estudo serão extraídos.
  • Recortar elementos-Só as porções dos elementos que estão dentro da área de trabalho vão ser extraídos.

Formato dos dados de saída

Formato dos dados que serão extraídos e descarregados. Os pacotes de camadas incluem sempre geodatabases.

Nome do ficheiro de saída

Este é o nome do item que será adicionado ao O Meu Conteúdo. O nome do item é baseado no Formato dos dados de saída. Se o item já existir, será questionado se pretende substituir o item existente.

Ao utilizar a opção Guardar resultado em pode definir o nome da pasta em O Meu Conteúdo onde o resultado será guardado.
