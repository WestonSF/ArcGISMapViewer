레이어 중첩

중첩은 둘 이상의 레이어를 하나의 단일 레이어로 결합합니다.맵 스택을 살펴보고 스택에서 확인한 모든 정보를 포함하는 단일 맵을 생성하는 작업을 중첩이라고 생각하면 됩니다.실제로 GIS가 등장하기 전에는 지도 제작자들이 깨끗한 아세테이트 시트에 맵을 복사하여 이러한 시트를 밝은 테이블 위에 포개 놓고, 포개진 데이터에서 새 맵을 손으로 그렸습니다.중첩은 단순히 라인 작업의 병합을 의미하는 것이 아닙니다. 중첩에 참가한 피처의 모든 특성이 최종 결과물까지 그대로 유지됩니다.중첩은 "무엇과 무엇이 겹쳐 있습니까?"라는 지오그래피의 가장 기본적인 질문 중 하나에 대답하는 데 사용됩니다.예를 들면 다음과 같습니다.

  • 100년 홍수 지역 내에 있는 필지는 무엇입니까?(여기에서'내에 있다'는 '겹침'을 의미하는 또 다른 표현 방법)
  • 어떤 도시 내에 어떤 도로가 있습니까?
  • 토질 유형별 토지 이용 현황이 어떻습니까?
  • 버려진 군사 기지에 어떤 우물이 있습니까?

현재 맵 범위 사용을 선택한 경우 현재 맵 범위 내에 표시된 분석 레이어와 중첩 레이어의 피처만 중첩되고,이 옵션의 선택을 취소한 경우 현재 맵 범위 외부에 있는 피처를 포함하여 분석 레이어와 중첩 레이어에 있는 모든 피처가 중첩됩니다.

중첩 레이어 선택

분석 레이어와 중첩할 레이어입니다.

중첩 방법 선택

중첩 방법은 분석 레이어와 중첩 레이어가 결합되는 방식을 정의합니다.

  • 교차 - 분석 레이어의 피처와 겹치는 중첩 레이어의 피처 또는 피처의 일부만 결과에 포함됩니다.분석 레이어가 영역 피처인 경우 중첩 레이어는 영역, 포인트 또는 라인 피처일 수 있습니다.분석 레이어가 라인 피처인 경우 라인 또는 포인트 피처만 중첩 레이어가 될 수 있습니다.분석 레이어가 포인트 피처인 경우에는 중첩 레이어도 포인트 피처여야 합니다.
  • 유니온 - 분석 레이어와 중첩 레이어의 피처가 모두 결과에 포함됩니다.유니온 옵션은 분석 레이어와 영역 레이어 둘 다 영역 피처가 포함된 경우에만 사용할 수 있습니다.
  • 지우기 - 분석 레이어의 영역과 겹치지 않는 중첩 레이어의 피처 또는 피처의 일부가 결과에 포함됩니다.중첩 레이어에는 영역, 라인 또는 포인트 피처가 포함될 수 있습니다.

현재 맵 범위 사용을 선택한 경우 현재 맵 범위 내에 표시된 분석 레이어와 중첩 레이어의 피처만 중첩되고,이 옵션의 선택을 취소한 경우 현재 맵 범위 외부에 있는 피처를 포함하여 분석 레이어와 중첩 레이어에 있는 모든 피처가 중첩됩니다.

결과 레이어 이름

내 컨텐츠에서 생성되어 맵에 추가되는 레이어의 이름입니다.피처 유형(영역, 라인 또는 포인트)은 중첩 레이어와 같습니다.기본 이름은 중첩 방법과 입력 레이어 이름을 기반으로 합니다.레이어가 이미 있는 경우 덮어쓸지 확인하는 메시지가 나타납니다.

결과 저장: 드롭다운 상자를 사용하여 내 컨텐츠에서 결과를 저장할 폴더의 이름을 지정할 수 있습니다.
