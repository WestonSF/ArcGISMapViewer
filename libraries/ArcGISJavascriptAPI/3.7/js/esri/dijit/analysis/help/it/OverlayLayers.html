Sovrapponi layer

Consente di combinare due o più layer in un unico layer. La sovrapposizione è un processo analogo all'analisi di un insieme di mappe e alla creazione di un unica mappa contenente tutte le informazioni trovate nel gruppo. Di fatto, prima dell'avvento del GIS, i cartografi copiavano letteralmente le mappe su lucidi, sovrapponevano i lucidi su un tavolo luminoso e disegnavano a mano una nuova mappa con i dati sovrapposti. La sovrapposizione non è una semplice unione di linee; nel prodotto finale vengono infatti inclusi tutti gli attributi delle feature coinvolte nella sovrapposizione. La sovrapposizione costituisce la risposta a una delle più elementari domande di geografia, ovvero "Cosa sta sopra e cosa sta sotto?" Ad esempio:

  • Quali lotti sono compresi nella pianura alluvionale da 100 anni? )
  • Quali strade sono comprese in determinate regioni?
  • Quale uso del suolo si fa di un determinato tipo di terreno?
  • Quali pozzi si trovano in basi militari abbandonate?

Se si seleziona Usa estensione mappa corrente, verranno sovrapposte solo le feature nel layer di analisi e di sovrapposizione visibili nell'estensione mappa corrente. Se questa opzione è deselezionata, verranno sovrapposte tutte le feature sia nel layer di analisi che in quello di sovrapposizione, anche se esterne all'estensione mappa corrente.

Scegli metodo di sovrapposizione

Il metodo di sovrapposizione definisce in che modo vengono combinati il layer di analisi e quello di sovrapposizione.

  • Incrocia: il risultato conterrà solo le feature o le parti di feature del layer di sovrapposizione che risultano sovrapposte a feature del layer di analisi. Se il layer di analisi include feature area, il layer di sovrapposizione può includere feature area, punto o linea. Se il layer di analisi include feature linea, il layer di sovrapposizione può includere solo feature linea o punto. Se il layer di analisi include feature punto, anche il layer di sovrapposizione deve includere feature punto.
  • Unione: il risultato conterrà feature sia del layer di analisi che di quello di sovrapposizione. Questa opzione è disponibile solo se il layer di analisi e quello di sovrapposizione contengono entrambi feature area.
  • Cancella: il risultato conterrà le feature o le parti di feature del layer di sovrapposizione che non risultano sovrapposte ad aree del layer di analisi. Il layer di sovrapposizione può contenere feature area, linea o punto.

Se si seleziona Usa estensione mappa corrente, verranno sovrapposte solo le feature nel layer di analisi e di sovrapposizione visibili nell'estensione mappa corrente. Se questa opzione è deselezionata, verranno sovrapposte tutte le feature sia nel layer di analisi che in quello di sovrapposizione, anche se esterne all'estensione mappa corrente.

Nome layer risultati

Questo è il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il tipo di feature del layer, ovvero area, linea o punto, è uguale a quello di sovrapposizione. Il nome predefinito è basato sul metodo di sovrapposizione e sul nome del layer di input. Se il layer esiste già, verrà chiesto se si desidera sovrascriverlo.

Utilizzare la casella a discesa Salva risultato in per specificare il nome di una cartella di I miei contenuti in cui salvare il risultato.
